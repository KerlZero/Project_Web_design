# Visual Mockups Version Process

This folder stores visual mockup versions for the Component Library project.

## Rule For Future Design Changes

Every time the design is changed, improved, or a new function is added, ask first:

```text
Do you want me to save this as a new visual mockup version and store the updated plan in the Visual-Mockups folder?
```

If approved, create a new version folder:

```text
Visual-Mockups/
  v1/
  v2/
  v3/
```

Each version folder should include:

- Visual mockup image
- Plan or change summary
- Any important design notes for that version
