# v1 Current Bugfix Scope

## Purpose

This file defines the active working scope for the current version.

## Active Mode

Work on the current version is limited to bugfixes only.

## Allowed Work

- Fix broken behavior in the current `SourceCode`.
- Fix layout issues that make the current v1 design unusable.
- Fix missing or incorrect integration of assets that already exist in the approved asset manifest.
- Add or update QA checks that validate the current v1 design contract.

## Not Allowed Without Project Owner Approval

- Implement backlog items from `FUTURE_FEATURES.md`.
- Add new product features.
- Change the approved visual direction.
- Update visual baselines unless UX/UI approval is explicit.
